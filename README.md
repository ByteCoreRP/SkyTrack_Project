# 🚁 SkyTrack Logistics - Dashboard de Drones

> Um sistema moderno para gerenciamento logístico de drones, focado em eficiência, usabilidade e visual profissional.

---

## 📌 Sobre o Projeto

O **SkyTrack Logistics** é um protótipo de sistema desenvolvido para a disciplina de Front-End, com o objetivo de simular um ambiente real de controle e monitoramento de drones.

A aplicação permite visualizar informações estratégicas, facilitando a tomada de decisão em operações logísticas.

---

## 🧠 Objetivo

Criar uma interface intuitiva e responsiva que funcione bem tanto em campo quanto em ambientes corporativos.

---

## 🛠️ Tecnologias Utilizadas

- 🌐 **HTML5 Semântico**
- 🎨 **CSS3**
  - Flexbox
  - Grid Layout
  - Variáveis CSS
- 📱 **Design Responsivo (Mobile First)**
- 🔄 **Git & GitHub (Versionamento)**

---

## 🚧 Desafios Superados

✔️ Implementação da **Tríade do Truncamento**  
✔️ Uso de **validação nativa de formulários** para maior segurança  
✔️ Desenvolvimento de **layout adaptável** (tablets e ultrawide)  

---

## 💡 Diferenciais do Projeto

- Interface limpa e moderna  
- Experiência do usuário otimizada  
- Estrutura de código organizada e escalável  
- Foco em boas práticas de desenvolvimento Front-End
